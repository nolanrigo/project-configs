# @nolanrigo/prettier-config

## Installation

```bash
$ npm i -D @nolanrigo/prettier-config
OR
$ yarn add --dev @nolanrigo/prettier-config
```

## package.json

```json
{
  "name": "my-package",
  ...
  "prettier": "@nolanrigo/prettier-config",
  ...
  "devDependencies": {
    "@nolanrigo/prettier-config": "^X.0.0"
  }
}
```
